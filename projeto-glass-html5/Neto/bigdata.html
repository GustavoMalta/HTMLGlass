TRABALHO SOBRE BUSINESS
INTELLIGENCE (B.I.) E DATAMINING
Desenvolvimento de uma apresentação de B.I. Business
Intelligence (Inteligência de negócios), que permita a análise de um volume
de dados fictício por diferentes perspectivas.
O projeto (código fonte) desenvolvido pelo aluno deverá ser
desenvolvido por ferramentas disponíveis no laboratório da escola, permitindo
a manutenção e consulta dos fontes e da base de dados utilizada pela
aplicação.
Criar um banco de dados com as tabelas necessárias ao projeto e
gerar dados de forma (aleatória/randômica) mas não linear. Sugere-se que a
tabela principal da base de dados tenha 30.000 a 40.000 registros.
Exportar os registros da tabela principal para um arquivo texto
(.txt) para que seja importado pelo Excel afim de permitir analisar os dados por
diversas perspectivas diferentes (Gráficos Dinâmicos).
A primeira parte do trabalho será entregue de forma textual
(formatação conforme padrão visto na disciplina de metodologia, capa,
espaçamento, fonte, etc) e deverá conter as seguintes informações abaixo
especificadas.


 Apresentar os objetivos do trabalho

* Desenvolvimento de aplicação com acesso a banco de dados
afim de contribuir com o aperfeiçoamento das técnicas de
programação e criação de sistemas.
* Motivação pela definição do tema
* Descrição do tema escolhido pelo aluno.
* Perspectivas a serem analisadas como resultado do projeto
* citar 5 perspectivas que se deseja representar
* Modelagem do banco de dados (DER) da aplicação proposta
* Ferramentas utilizadas para o desenvolvimento
* Fontes bibliográficas utilizadas


Cronograma:
1a. aula: Apresentação detalhada da proposta de trabalho a ser
desenvolvida (descrição do objetivo proposto)
* Objetivo a ser alcançado
* Motivação pela escolha do tema
* Perspectivas a serem apresentadas como resultado
2a. aula: Pesquisa sobre conceitos elementares sobre BI
* Apresentar o conceito e finalidade dos termos abaixo:
* Business Intelligence - inteligência nos negócios
* Data WareHouse - depósito de dados.
* Data Mining - mineração de dados
* OLAP
* OLTP
* Relatar um caso de uso real. Empresa que tenha utilizado ou faça
uso destas tecnologias (ou alguma delas) e o resultado obtido,
informar o link e ou origem desta informação.
3a. aula: Entrega da primeira parte do material escrito (realizado nas duas
aulas anteriores) respeitando formatação de trabalho científico. Início do
desenvolvimento do projeto. Montar o DFD geral do projeto. Identificando as
entidades envolvidas.
4a. aula: Modelagem.
* Apresentar a modelagem do projeto (DER) e início do
desenvolvimento em laboratório do software para geração dos dados
fictícios para popular a tabela principal do banco de dados.
5a. aula: Gerar arquivo texto para importação.
* Finalização da geração de dados na tabela principal do banco de dados,
gerar o arquivo texto para ser importado pelo Excel. (Pode-se utilizar
qualquer outro método para exportar para o Excel).
6a. aula: Apresentação da aplicação de BI desenvolvida, bem como o
material textual terminado com informações sobre ferramentas e
bibliografias utilizadas.
